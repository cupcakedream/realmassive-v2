Divi Blog Extras is a plugin for Divi theme which include pre-build blog category layouts.

== Description ==
Divi Blog Extras is a plugin is a plugin by Elicus Technologies that adds a module which allows Divi WordPress theme uses to get pre-built blog category layouts.

== Installation ==
Upload the divi-blog-extras folder to the /wp-content/plugins/ directory.
Activate the plugin through the �Plugins� menu in WordPress.

Or

Upload the zip file from WordPress dashboard.
Once installed, activate the plugin
Go to Page/Post and create a section.
Click on Insert module and then select \\\"Divi Blog Extras\\\".
Finally select the parameters like type of layout out of available layouts, number of blogs to show, categories, featured images and more like the standard Divi blog module.
